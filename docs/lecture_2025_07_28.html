인터랙티브 교수-학습 웹 앱

  • 학습자와 콘텐츠 간의 상호작용을 중심으로 설계되어, 능동적 참여, 즉각적 피드백, 탐구 활동 등을 통해 개인의 학습 경험을 향상시키는 교육용 웹 애플리케이션

  • 성격

    • 디지털 학습지: 교과의 학습 내용을 효과적으로 교수/학습하도록 도와주는 도구

    • 탐구 학습 도구: 데이터 탐색을 통해 학습자 스스로 이해, 지식, 통찰을 얻도록 도와주는 도구

인터랙티브 교수-학습 웹 앱

  • 웹 브라우저를 통해 접근하고 사용하는 교수-학습 목적의 애플리케이션

  • 데이터 대시보드: 단일한 주제에 대한 상호연관된 다양한 데이터 정보를 그래픽의 형태로 일관성 있게 제시한 것

    • 레이아웃 요소: 카드(행과 열), 페이지, 내비게이션바, 사이드바, 툴바, 탭셋

    • 내용 요소: 텍스트, 표, 그래프, 지도, 동영상, 밸류 박스

    • 작동 요소: 정적, 동적, 인터랙티브

TPACK-DBL 프레임워크

TPACK-DBL

  • 디지털 제작 도구에 대한 이해와 활용 능력(디지털 역량)

  • 복잡한 개념을 시각적이고 구조적으로 전달하는 능력(커뮤니케이션 역량)

  • 분석적 사고 및 데이터 기반 수업 구현 능력(데이터 리터러시 + 기술력)

  • 오픈 지식 생태계에의 기여(공유 및 협업 역량)

제작 도구

  • Quarto(쿼토): 대시보드 제작 및 웹 앱 생성

    • “과학적, 기술적 출판을 위한 위한 오픈소스 시스템”

    • 다양한 형식의 저작물(노트, 연구 논문, 프레젠테이션, 대시보드, 웹사이트, 블로그, 서적 등)을 다양한 디지털 포맷(HTML, PDF, MS Word, ePub 등)으로 출판할 수 있게 해주는 도구

  • R(혹은 Python): 인터랙티브 콘텐츠 제작

    • 데이터사이언스 프로세스의 실행

    • 인터랙티브 지리공간 시각화 및 탐색 기능의 구현
